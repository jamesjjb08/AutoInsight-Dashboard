# Automated Multi Dataset Insight Dashboard

This project automatically analyzes uploaded datasets and generates a live interactive dashboard.

Features:
- Upload CSV or Excel datasets
- Automatic data type detection
- Automatic chart generation
- Interactive dashboard
- Correlation analysis

Technologies Used:
- Python
- Streamlit
- Pandas
- Plotly

Run the project:

pip install -r requirements.txt

streamlit run auto_dashboard.py
